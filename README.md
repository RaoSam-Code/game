# 🕵️ Midnight at Blackwood Manor

An AI-powered voice detective mystery game where every conversation matters.

**Interrogate AI suspects**, watch their emotions change in real-time, and solve a procedurally generated murder mystery.

## Features
- 🎤 Voice interrogation via Web Speech API
- 🤖 AI suspects powered by Groq (Llama 3.3 70B)
- 🔄 New mystery every game — procedurally generated
- 😰 Dynamic emotional expressions
- 🔍 Automatic evidence collection
- 💡 Progressive hint system
- 🔊 Immersive procedural audio

## Setup
Set `GROQ_API_KEY` as a Space Secret.
