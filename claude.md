## 🎯 **PROBLEM STATEMENT**

### **Current Gaming Landscape Issues:**

1. **AI in Games is Scripted** 
   - NPCs have pre-written dialogue trees
   - Same responses every playthrough
   - No real "intelligence"

2. **Detective Games Lack Immersion**
   - Click through text
   - No voice interaction
   - Static character expressions
   - Feels like reading, not investigating

3. **No Real Consequence**
   - Can't actually "catch" suspects in lies
   - Predetermined story paths
   - Player agency is illusion

### **What We're Solving:**
- ✅ **Real AI conversations** - No scripts, genuine interrogation
- ✅ **Voice-first gameplay** - Speak naturally like a real detective
- ✅ **Dynamic emotional responses** - Suspects react to YOUR questions
- ✅ **Multiple solution paths** - Different strategies work
- ✅ **Actual challenge** - AI suspects lie convincingly

## 💡 **THE SOLUTION**

### **What Is "Midnight at Blackwood Manor"?**

An **immersive detective mystery game** where:

**🎭 4 AI-Powered Suspects**
- Each with unique personality, backstory, and secrets
- Powered by Claude AI - they remember everything you ask
- They lie, deflect, get nervous, get angry
- One of them is the killer

- Speak your questions naturally (or type)

- Different voices for each character

**😰 Emotional Intelligence**
- Facial expressions change dynamically:
  - 😰 Nervous when lying
  - 😠 Angry when accused

### **2. Dynamic Emotional Expressions** 😰

**Expression System:**
```javascript
const EXPRESSIONS = {
  neutral: { victoria: '👩‍💼', james: '🤵', eleanor: '👩‍⚕️', margaret: '👵' },
  nervous: { victoria: '😰', james: '😅', eleanor: '😬', margaret: '😟' },
  angry: { victoria: '😠', james: '😤', eleanor: '😡', margaret: '😦' },
  happy: { victoria: '😊', james: '😎', eleanor: '🙂', margaret: '😌' },
  thinking: { victoria: '🤔', james: '🤨', eleanor: '🧐', margaret: '🤔' }
};

// Auto-detect mood from AI response
function analyzeMood(text) {
  if (text.includes('nervous') || text.includes('...')) return 'nervous';
  if (text.includes('!') || text.includes('how dare')) return 'angry';
  if (text.includes('glad') || text.includes('relieved')) return 'happy';
  if (text.includes('recall') || text.includes('think')) return 'thinking';
  return 'neutral';
}

// Expression changes in real-time during conversation
// Resets to neutral after 5 seconds
```

**Player Impact:**
- **Visual lie detection** - Nervous = hiding something
- **Emotional manipulation** - Make them angry to slip up
- **Non-verbal cues** - Just like real interrogation
- **Immersion** - Characters feel alive

### **4. Evidence Auto-Detection** 🔍

**Smart Clue System:**
```javascript
// Analyzes conversation for keywords
function detectEvidence(question, response, suspect) {
  const keywords = {
    'affair': `${suspect.name}: Mentioned affair`,
    'alibi': `${suspect.name}: Provided alibi`,
    'poison': `${suspect.name}: Discussed poison access`,
    '9:45': `${suspect.name}: Timeline at 9:45 PM`,
    'study': `${suspect.name}: Was near the study`,
    'blackmail': `${suspect.name}: Blackmail mentioned`,
    'saw': `${suspect.name}: Witnessed something`
  };
  
  const combined = (question + ' ' + response).toLowerCase();
  
  // Check all keywords
  Object.entries(keywords).forEach(([key, evidence]) => {
    if (combined.includes(key) && !evidenceBoard.includes(evidence)) {
      addEvidence(evidence);
      playSound('evidence');  // 🎵 Chime
      showToast(`New Evidence Found!`);
    }
  });
}
```

**Player Benefits:**
- Don't miss important clues
- Visual progress tracking
- Satisfaction of discovery
- Helps less experienced players

### **6. Smart Hint System** 💡

**Dynamic Hints Based on Progress:**
```javascript
function getHint() {
  const questionCount = getTotalQuestions();
  const evidence = getEvidenceBoard();
  
  // Progressive hint logic
  if (questionCount < 3) {
    return "💡 Start with Margaret (housekeeper) - she sees everything";
  }
  
  if (!evidence.includes('9:45')) {
    return "💡 Ask suspects where they were at 9:45 PM";
  }
  
  if (!evidence.includes('Margaret') && !evidence.includes('saw')) {
    return "💡 Ask Margaret if she saw anything suspicious";
  }
  
  if (evidence.includes('Hayes') && evidence.includes('9:45')) {
    return "💡 You have key evidence! Confront Dr. Hayes about her alibi";
  }
  
  return "💡 Look for contradictions in the timelines";
}
```

**Non-Intrusive:**
- Collapsible hint panel
- Only shows when player clicks
- Doesn't spoil the solution
- Guides without hand-holding
